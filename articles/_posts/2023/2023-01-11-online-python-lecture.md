# 특징

* 풀 인터랙티브 : 브라우저 내에서 코드 수정하고 답변 작성
* 쉬운 디버깅 기능 및 REPL 내장 (Pyodide 이용)
* 에러에 대한 상세 도움말 제공
* 문제를 못 풀 경우 힌트를 제공하고 그래도 못풀면 답안 순서를 조정하는 형태로 추가 힌트를 제공
